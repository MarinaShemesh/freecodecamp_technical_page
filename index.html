Introduction

We all start our coding journey with the index.html page.

And though it seems so exciting to see that first "Hello world!" message on your browser, remember that this is merely a very small step into your coding journey.

Code Editor

To write code you are going to need some type of code editor, for example Notepad ++, Atom, Sublime Text or Brackets. Do your research and decide which want suits you and then install it.

My personal favourite is Sublime Text.

Hello World

Now let's get cracking with writing our first code!

Open your code editor, copy this boilerplate code:

 <!DOCTYPE html>
  <html lang="en">
   <head>
    <title>
    </title>
   </head>
   <body>
   </body>
  </html>

Between the two body tags, write warp "Hello world" in h1 tags like this:

  <H1> Hello World! </H1>

Other headings, in descending degrees of importance are written as:

  • h2
  • h3
  • h4
  • h5
  • h6

Open Browser

Check the code that you have written by saving your file as index.html

Navigate to this file where you have saved it and double-click onit. You will see that a new tab in the browser opend up with an empty white page with the words "Hello World!".

Add CSS

Let's change the colour of "Hello world" by creating as css file. In the same folder as the index.html create this file:

 style.css

Make sure that your index.html file knows about the css file by adding this link in the head part of the index.html file:

  <link rel="stylesheet" type="text/css" href="style.css">

Add the following code to your css file, refresh the page and see how the "Hello World!" message magically change color and font:

 h1{
 font-family:"Arial";
 color:red;
 }
